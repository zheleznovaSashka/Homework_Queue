\# Очереди и управление пользователями



\## Описание проекта

Проект содержит три задания по работе с очередями и структурами данных.



\## Задания



\### Задание 1. Обычная очередь

Реализация очереди для работы с символьными значениями.

\- \*\*IsEmpty\*\* - проверка на пустоту

\- \*\*IsFull\*\* - проверка на заполнение  

\- \*\*Enqueue\*\* - добавление элемента

\- \*\*Dequeue\*\* - удаление элемента

\- \*\*Show\*\* - отображение всех элементов



\### Задание 2. Очередь с приоритетами

Реализация очереди с приоритетами на основе узлов.

\- \*\*InsertWithPriority\*\* - добавление элемента с приоритетом

\- \*\*PullHighestPriorityElement\*\* - удаление элемента с высшим приоритетом

\- \*\*Peek\*\* - просмотр элемента с высшим приоритетом

\- \*\*IsEmpty\*\*, \*\*IsFull\*\*, \*\*Show\*\*



\### Задание 3. Управление пользователями

Реализация на основе связного списка.

\- Добавление нового пользователя

\- Удаление пользователя

\- Проверка существования пользователя

\- Изменение логина

\- Изменение пароля
